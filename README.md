## **ðŸ“¦ Installation**  
1. **Clone the repo**:  
   ```bash
   git clone https://github.com/your-username/your-repo.git
   cd your-repo
   ```

2. **Set up the backend**:  
   ```bash
   cd server
   npm install
   echo "MONGODB_URI=your_mongodb_connection_string" > .env
   npm start
   ```

3. **Set up the frontend**:  
   ```bash
   cd ../client
   npm install
   npm start
   ```

4. **Open the app**:  
   Visit `http://localhost:3000` in your browser.
